# Omnecor Help & Onboarding Guide

Welcome to **Omnecor HMCI**. This guide provides essential information to get started.

## Quick Start

1. **Launch Omnecor**: `npm run dev`
2. **Access Dashboard**: Open `http://localhost:3000`
3. **Explore Features**: Review the User Guide (`USER_GUIDE.md`)

## Frequently Asked Questions

### What is Omnecor?

Omnecor is the premier Human-Machine Collaboration Interface, acting as the central nervous system for digital and physical workflows.

### Is my data private?

Yes, Omnecor operates in **Zero-Login mode** by default. Your data remains local.

### Where can I find support?

- **Troubleshooting Guide**: `TROUBLESHOOTING.md`
- **GitHub Issues**: Report bugs or request features.
- **Support Email**: support@omnecor.ai
