## Getting Started

### Dashboard

The dashboard provides a unified view of your projects, active models, and recent activities.

### Zero-Login Mode

By default, Omnecor prioritizes local data storage, ensuring your projects and configurations remain private.

## Neural Workspaces

Visualize your project structure, knowledge bases, and AI agent relationships using interactive neural network graphs.

## Chat Interface

Interact with a wide range of AI models using a unified, context-aware chat interface with real-time streaming and token tracking.

## Model Hub

Manage and configure local AI models (Ollama) and cloud AI providers (OpenAI, Anthropic, Gemini) in a centralized hub.

## Specialized Modules

Omnecor natively supports advanced engineering tools:

- **Blender Bridge:** 3D model generation and modification.
- **KiCad Bridge:** PCB schematic and layout design.
- **ESPTool Bridge:** Hardware flashing and communication.

## Integrations

Connect external services (GitHub, Notion, Slack) to enrich your project's knowledge base.

## Settings

Configure workstation preferences, security settings, API keys, and model performance parameters.
